# skip-intro-sync
contributes `skip_first` values on studios up to [skips-db](https://github.com/feederbox826/skips-db).

This is not needed for any skipping, **only** for contributing times from [skip-intro](https://discourse.stashapp.cc/t/skip-intro/128) back to the database. See [skips-db](https://github.com/feederbox826/skips-db/blob/main/API.md#identifiers-privacy) for privacy info.
